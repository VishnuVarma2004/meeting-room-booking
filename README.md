#  Office Meeting Room Booking System

> A full-stack web application to manage and book meeting rooms efficiently with real-time availability and conflict prevention

##  Live Features

-  Book meeting rooms with selected time slots  
-  View available rooms in real-time  
-  Prevent duplicate bookings  
-  Automatically handle expired bookings  
-  Simple and clean user interface  

---

##  Project Overview

Managing meeting rooms manually can be confusing and often leads to double bookings.

This project solves that problem by providing a centralized system where users can easily check availability and book rooms without conflicts.
